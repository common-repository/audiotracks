Manage a directory of audio tracks in WordPress. 

== Description ==

At the moment this plugin simply allows you to create and manage a director y of audio tracks in WordPress.

Future plans include:

* Additional track details; Lyrics, Year, BPM, track numbers etc
* Additional audio file information; Bit rate, Sample rate etc
* Audio player functionality

== Installation ==

Installation is very simple:

1. Copy/upload the `audiotracks` folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
